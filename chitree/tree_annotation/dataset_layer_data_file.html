Dataset layer data file

Video tutorial

First column

The first column of each dataset layer data file must be node IDs which are identical with tree file. You can copy all node IDs of the tree to the system clipboard for easy inclusion in layer data files by clicking button copy-nodesID in tree file menu. In most cases, node IDs are identical with sequence IDs in the sequence file like fasta format when building phylogenetic tree. The order of node IDs in the first column can be random, but row order of data in other columns must keep corresponding to each node ID. The column name of each column can be customized.

Other columns

Data in other columns depends on which dataset layer you want to add, which is described in more detail in each dataset layer below.

File format

Dataset layer data file can be in one of supported formats (Microsoft Excel, TSV, CSV).
